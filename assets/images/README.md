# 图片文件夹

## 文件夹结构
```
images/
├── README.md          # 本文件
├── photos/            # 摄影作品（请上传你的照片）
│   └── README.md
├── projects/          # 项目截图
│   └── README.md
└── favicon.ico        # 网站图标（可选）
```

## 添加照片

1. 将你的摄影作品上传到 `photos/` 文件夹
2. 支持格式：jpg, png, webp
3. 建议尺寸：宽度至少 800px
4. 重命名为有意义的名称，如：`shanghai-night-01.jpg`

## 添加项目截图

1. 将你的 AR/Unity 项目截图上传到 `projects/` 文件夹
2. 建议尺寸：16:9 比例
3. 重命名为项目名称，如：`ar-game-demo.jpg`

## 注意事项

- GitHub 仓库单文件限制 25MB
- 建议使用图床服务托管大图
- 推荐使用 WebP 格式压缩图片
