# 简历 PDF

请将你的简历 PDF 文件上传到这个文件夹，命名为 `resume.pdf`。

## 文件要求
- 格式：PDF
- 文件名：`resume.pdf`
- 建议大小：不超过 2MB

## 更新网站中的下载链接

修改 `index.html` 中简历区域的下载按钮：
```html
<a href="./assets/resume/resume.pdf" class="btn btn-outline download-btn" download>
    <i data-lucide="download"></i>
    <span>下载 PDF</span>
</a>
```

确保路径正确即可。
